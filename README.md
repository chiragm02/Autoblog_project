# AutoBlog Concierge
Automatically generates researched, SEO-optimized blog posts using a multi-agent Python system.
build a multi-agent system that turns a topic brief into a researched, SEO-optimized draft and publishes a formatted post to a GitHub Pages / static site. Demonstrates multi-agent design (parallel + sequential), tools (web search, code-exec for formatting, OpenAPI), sessions & memory, context engineering, observability, and evaluation
